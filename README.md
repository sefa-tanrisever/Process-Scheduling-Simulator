Process Scheduling Simulator
This project is a comprehensive simulation of CPU Scheduling Algorithms developed as part of an Operating Systems study. It visualizes and calculates key performance metrics for various process management strategies.

Algorithm Implementations: * FCFS (First-Come, First-Served): Simplest scheduling, non-preemptive.
SJF (Shortest Job First): Optimized for minimum average waiting time.
Round Robin (RR): Time-sharing system with a specific time quantum.
Priority Scheduling: Executing processes based on assigned priority levels.
Performance Metrics: Automatic calculation of Average Waiting Time (AWT) and Average Turnaround Time (ATAT).
Gantt Chart Visualization: Clear representation of the execution sequence.

Tech Stack
Language: Python

Concepts: Process Management, Context Switching, CPU Scheduling.
